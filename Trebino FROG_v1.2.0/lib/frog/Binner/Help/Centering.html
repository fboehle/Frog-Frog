Centering Panel

The centering panel allows the selection of several centering algorithms.

The Centering Panel
Delay/Frequency Centering
The centering algorithm to use in the binning step.
Delay/Frequency Center
An indicator/control showing the center value of the trace.

The centering algorithms include:

None
No centering
First Moment
Use the first moment of the marginal as the center.
Maximum
Use the maximum of the marginal as the center.
Manual
Use the value in the text box.

In FROG the delay should be centered (either first moment or maximum), while, in XFROG, centering is unnecessary. In all cases (at the time of this writing) frequency centering is not needed, although it can be done.
