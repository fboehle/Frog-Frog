Calibration Panel

The calibration panel allows setting the delay spacing, wavelength spacing, and center wavelength. These values are simply entered in their respective text boxes.

Currently the values of the nonlinearity and FROG type boxes are ignored.

If there was a valid header in the file loaded the values for the spacing should have been set properly.

The Calibration Panel
Delay Spacing
The delay spacing of the measured FROG trace. This value is determined from calibrating the FROG device.
Wavelength Spacing
The wavelength spacing.
Center Wavelength
The wavelength of the center pixel (N/2+1).
Nonlinearity
Currently unused.
FROG Type
Currently unused.
