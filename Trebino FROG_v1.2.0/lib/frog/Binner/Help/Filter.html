Filter Panel

This panel allows filtering out of high frequency noise. It currently uses a simple lowpass filter; simply eliminate frequencies above the cutoff.

The Filter Panel.

Filter Cutoff (%)
The cutoff percentage is the percentage of the existing frequencies to eliminate.
Filter Delay/Frequencies
These buttons filter the appropriate dimension, using the cutoff specified above.

Note: I do not like to filter FROG traces. I feel that the FROG algorithm does a fine job all on its own. Use filtering at your own risk.
