Extraction Panel

The extraction panel allows the removal of pesky, offending artifacts.

This panel can be used to remove unnecessary point from a trace. Care should be exercised. DO NOT remove valid data. Only remove noise.

The Extraction Panel: Area selected.

The first step in extraction is to select the area with the mouse. To change the selected area just reselect it with the mouse. The area can also be manually set with the text controls.

Delay/Wavelength Low/High
The limits of the selected area of the FROG trace.
Extract Data
Crops the trace to the size of the selected area.
Erase Data
Erases the selected area. Be careful.
The Extraction Panel: Area ERASED!! OOOPS!!!.

Be careful with the erase button. It can be dangerous. Note the caution above the button.

In general the erase button should NOT be used. It is intended for those rare cases where true artifacts show up in the data.
